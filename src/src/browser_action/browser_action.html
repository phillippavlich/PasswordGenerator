Here is what our app does and how it works:

  • You (the user), chooses a key (you will need to remember this)
  • The system uses the key provided the user, the domain of the webpage, and a salt(random string) stored locally
  • The program concatenates the inputs from the step above and hashes the result several times
  • The options page (settings button) allows you to customize the srequirements for your password
  • The system returns a secure password that matches the preferences
  • This password can be used for the current website.
  • Memorization is not important as the password will always be the same when given the same inputs.
